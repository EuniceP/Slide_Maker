
# Slide_Maker

####What is Slide Maker?
```
Slide Maker is a web application that allows you to upload images on line for you to create slides that run at specified interval.  

Think wedding/birthday/graduation/honeymoon picktures and slides that you can show off to your friends and family.
```
####Features
```
1. Login/password authentication
2. Upload/remove image files (.jpg, .png)
3. Allow users to sequence the images files - drag & drop image thumbnails from selected directory
4. Available files to sequence will be shown as thumbnails at bottom or right  of the screen and the sequence panel will be at the top/left of the screen
5. Allow to save slide object containing sequenced file names in a play list
6. Allow to create/update multiple slide objects and play lists
```
